# 1. Installation 

Get the specefic directory of cache into your !!

```bash
git clone github.com/mrinalxdev/time-cache.git

cd time-cache

rm -rf frontend

go mod tidy
```

# 2. Basic Usage 

```go
//main file
import (
    "time"
)

// Create a new cache with a 5-minute TTL
cache := timecache.NewCache(5*time.Minute, nil)

// Set a value
cache.Set("key", "value")

// Get a value
value, found := cache.Get("key")

// Remove a value
cache.Remove("key")
```
